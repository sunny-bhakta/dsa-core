## Overview
String algorithms deal with sequences of characters and pattern matching problems.

## Key Concepts

### String Operations
- Concatenation, substring, comparison
- Pattern matching
- String transformation

### Pattern Matching Algorithms
- **Naive**: O(nm) - check all positions (Implemented)
- **KMP**: O(n + m) - use failure function (Implemented)
- **Rabin-Karp**: O(n + m) average - rolling hash (Implemented)
- **Boyer-Moore**: O(nm) worst, O(n) best - skip characters (Implemented)
  - Bad character rule
  - Good suffix rule (Enhanced version)
